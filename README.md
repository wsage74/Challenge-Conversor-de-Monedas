# Conversor de Moneda - Challenge Alura ONE

Este proyecto fue desarrollado como parte del **Challenge de Alura Latam - Oracle Next Education (ONE)**.

Se trata de un **conversor de monedas en Java** que permite consultar tasas de cambio en tiempo real utilizando la API pública de ExchangeRate.

---

## Funcionalidades

- Conversión de COP → USD
- Conversión de USD → COP
- Conversión de COP → CLP
- Conversión de CLP → USD
- Conversión de COP → ARS
- Conversión de ARS → USD
- Consumo de API REST
- Parseo de respuestas JSON con Gson

---

## Tecnologías Utilizadas

- Java
- ExchangeRate API
- Gson 2.13.2

---

## API Utilizada

Página oficial:

https://www.exchangerate-api.com/

### Configuración de API Key

1. Crear una cuenta.
2. Generar tu API Key.
3. Colocar la clave en la variable `API_KEY` del proyecto.

En el Main.java encontrarás:

```java
private static final String API_KEY = "";
```
